# Comparación Programación Tradicional vs POO

Este proyecto calcula el promedio semanal de temperaturas.

## Programación Tradicional
Se usan funciones para ingresar datos y calcular el promedio.

## Programación Orientada a Objetos
Se utiliza una clase Clima con métodos para ingresar temperaturas y calcular el promedio.

Ambos programas resuelven el mismo problema usando diferentes paradigmas.
